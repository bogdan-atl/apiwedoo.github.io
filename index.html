    Introduction

    Welcome to the ServerRoutes API documentation. Here you'll find information on all the available endpoints and how to use them.

    Authentication

    Reset Password Request

    HTTP Request

    POST /api/v1/authentication/reset-password/request

    curl "http://ip/api/v1/authentication/reset-password/requsts/token"
    curl http://ip/api/v1/authentication/reset-password/requsts/token

    var xhr = new XMLHttpRequest();
xhr.open('POST', 'http://ip/api/v1/authentication/reset-password/request', true);

xhr.onload = function() {
  if (xhr.status === 200) {
    console.log(xhr.responseText);
  } else {
    console.log('error');
  }
};

xhr.send('token');

    The above command returns JSON structured like this:
